# Riesgo de Impago – Proyecto de Clasificación

**Descripción**  
Proyecto de análisis y modelado de riesgo de impago de clientes en una cooperativa financiera, desarrollado durante el curso de Minería de Datos y Aprendizaje Automático del programa de Estadística de la Universidad del Valle.  

Incluye:  
- `data/Data_Clientes_Cooperativa.txt`: datos crudos (4 117 observaciones y 11 variables).  
- `src/Script_Riesgo_Clientes.R`: script en R con todo el flujo de trabajo (preprocesamiento, EDA, modelado y evaluación).  
- `reports/Reporte_Riesgo_de_impago.pdf`: informe final en PDF.
